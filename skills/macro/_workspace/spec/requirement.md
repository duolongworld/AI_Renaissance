# 宏观分析框架 Skill（7层流水线）

> 本文档基于《宏观组分析框架手册 V3.1》编写，用于指导宏观分析 Skill 的开发。
> 
> **V3.1主要变更**：新增Layer 4.5（反身性与元认知层），聚焦AI科技主线，新增反身性压力计、逻辑生命周期监控、范式稳定性监测、人类裁决接口。
> 
> **重要**：一切从确切信息出发，不擅自编造方案内容。无法确定的内容已标出，需专家4组确认。

---

## 1. 适用范围

### 1.1 核心功能
基于**五大底层公理**，通过**7层流水线**（Layer 0 → Layer 5，新增Layer 4.5反身性与元认知层）的标准化分析体系，聚焦AI科技主线，从双经济体追踪、状态识别、周期定位、枢纽变量分析、市场定价提取、预期差信号引擎、反身性与元认知修正，到最终的 Beta+Alpha 资产配置，构建从宏观状态识别到资产配置的完整信号生产线。

### 1.2 适用场景
- 全球与中国宏观周期定位（增长/通胀/流动性三维）
- 资产配置决策（A股/美股/债券/商品/黄金）
- 行业轮动配置（申万一级行业/美股GICS行业）
- 预期差信号生成（数据意外/基本面vs市场定价/跨国预期差）
- 异常场景处理（黑天鹅/政策突变/数据修正/流动性危机）

### 1.3 分析对象
- 全球宏观经济变量（美联储政策/美债收益率/美元指数/VIX等）
- 中国宏观经济变量（PMI/社融/信贷/通胀/政策利率等）
- A股大类资产（沪深300/中证500/国债/商品/黄金）
- 美股大类资产（标普500/纳斯达克/美债/美元资产）
- 行业板块（A股申万一级/美股GICS一级）

### 1.4 不适用的场景
- 不提供具体个股买卖建议
- 不预测具体点位
- 不替代人工深度研究
- 不用于高频交易（信号有效期最短为1周）

### 2.3 缺失处理

| 缺失情形 | 处理规则 | 输出影响 |
|---|---|---|
| 单个指标临时缺失（如PMI尚未发布） | 使用过去3个月均值填充，标注 `uncertainties` | 信号置信度下调0.1 |

| 数据源完全中断（如Wind/ Bloomberg故障） | 启动备用数据源（如改用NBS/央行官网手动录入），标注 `evidence.source_type = "manual_backup"` | 信号置信度下调0.2 |
| 低频数据尚未发布（如季度GDP） | 使用高频指标插值估算，标注 `uncertainties` | 仅用于趋势判断，不用于精确点位 |

## 3. 分析步骤

> 本 Skill 采用**7层流水线**架构，各层按顺序执行，前一层输出作为后一层输入。
> 
> **V3.1新增Layer 4.5**：反身性与元认知层，监测共识拥挤程度和框架范式稳定性，对信号强度进行三重修正。

### 3.1 Layer 0：双经济体追踪与中美交互通道
**目标**：并行追踪中美五大维度指标，识别6条传导通道的触发状态。

**步骤**：
1. 抓取中美两国五大维度（增长/通胀/政策/流动性/市场定价）的最新数据
2. 对每个指标计算5年滚动z-score标准化
3. 计算跨国差值/比值（如中美10Y利差、中美CAI差值）
4. 检查6条传导通道是否触发（利差→资本流、美元→大宗、美联储→风险偏好、中国信贷→全球大宗、全球PMI→中国出口、地缘政治→风险溢价）
5. 判定交互层次（对称交互/非对称传导/反馈循环）

**输出**：两国5维度指标面板、跨国差值/比值信号、6条通道触发状态、交互层次判定

### 3.2 Layer 1：状态识别——CAI/FCI 合成指标
**目标**：将多指标合成为可跨期可比的标准化分数，回答"经济现在好不好"和"金融环境松还是紧"。

**步骤**：
1. 按权重表合成中国版CAI（22个指标加权平均）
2. 按权重表合成中国版FCI（5个指标加权平均）
3. 按权重表合成美国版CAI（9个指标加权平均）
4. 按权重表合成美国版FCI（5个指标加权平均）
5. 独立计算通胀得分（水平+动量双维度合成，不纳入CAI/FCI）

**输出**：中美CAI/FCI/通胀的z-score分数与方向标签

### 3.3 Layer 2：周期定位——4象限 + 政策维度 + 长期债务周期
**目标**：将连续分数映射为离散的周期状态，指导资产配置。

**步骤**：
1. 根据CAI和通胀得分，将中美分别定位到4象限（复苏/过热/滞胀/衰退）
2. 计算中国政策维度综合得分（货币政策0.4 + 财政政策0.3 + 地产政策0.25 + 监管事件0.05）
3. 根据政策得分调节4象限信号强度（> +0.5 → +1档；< -0.5 → -1档）
4. 判定长期债务周期位置（美国：末端/中段/早期；中国：下行/稳定/上行）
5. 执行周期一致性校验（全球周期优先、恐慌状态直接下调、中美分化仅输出结构性信号）

**输出**：中美4象限定位、政策维度调节、长期债务周期定位、周期一致性校验结果

### 3.4 Layer 2.5：枢纽变量分析层
**目标**：显式建模汇率与大宗商品的传导作用，连接中美宏观。

**步骤**：
1. 分析USD/CNH汇率方向（利差驱动0.3 + 经常账户0.2 + 风险偏好0.2 + 政策意图0.3）
2. 计算6个商品比值信号（铜金比、油金比、铁矿石/铜比、大豆/玉米比、黄金vs实际利率、南华vs全球PMI）
3. 定位全球宏观三角（全球紧缩/宽松/滞胀/通缩）
4. 检查关键传导通道警报

**输出**：USD/CNH方向得分、6个商品比值信号、全球宏观三角定位、关键传导通道警报

### 3.5 Layer 3：市场定价提取
**目标**：从市场价格反推市场已定价了什么，与Layer 1的实际基本面状态形成对照。

**步骤**：
1. 从利率市场反推增长/通胀/政策预期（期限利差、政策利率预期、隐含通胀）
2. 从估值反推盈利预期（前瞻PE历史百分位、股权风险溢价、周期股/防御股相对估值、A-H溢价）
3. 从汇率反推资本流动预期（USD/CNH 1Y远期点、CNH-CNY价差、人民币期权波动率微笑）
4. 从商品反推全球需求预期（铜金比、远月/近月期货价差、南华vs全球PMI隐含值）
5. 从信用反推风险偏好（AA信用利差、城投债-国开债利差、高收益债vs投资级利差）

**输出**：5类指标的中国/美国隐含预期得分，与Layer 1实际状态得分并列对照

### 3.6 Layer 4：预期差信号引擎
**目标**：计算"实际状态vs市场定价"的偏差，生成可执行信号。

**步骤**：
1. 跟踪高频意外指数（中国CESI、美国Citi Economic Surprise Index）
2. 计算基本面vs市场定价预期差（增长预期差、通胀预期差、政策预期差、外部预期差）
3. 计算跨国预期差（中美增长/通胀/政策/流动性/信用周期差、美元-大宗联动信号）
4. 对 each 信号计算强度评分（历史分位）和衰减调整后强度
5. 执行信号复核规则（连续2周反向→触发复核；连续3周反向→强制降级）

**输出**：3类预期差信号方向与强度、信号相关性矩阵、信号复核记录、不相关信号源筛选结果

### 3.7 Layer 4.5：反身性与元认知层（V3.1新增）
**目标**：监测共识拥挤程度和框架范式稳定性，对信号强度进行三重修正。

**设计依据**：
- 基于索罗斯的反身性理论：市场参与者的行为本身也改变市场
- 当越来越多资金采用相似的宏观分析框架和信号时，这些信号本身可能改变市场的运行方式
- 本层为框架装上一层"元认知"，回答"当前框架的信号是否仍然可靠？"

**三层元认知功能**：
1. **反身性压力计**：信号本身是否因共识拥挤而失效？
2. **范式稳定性监测**：当前环境是否仍在框架公理的适用范围内？
3. **逻辑生命周期监控**：当前主导逻辑处于哪个生命周期阶段？

**反身性压力计（4.5.2）**：
- 四大压力指标：信号拥挤度、仓位集中度、信号自我实现指数、跨框架一致性
- 压力计合成：反身性压力综合得分 = 信号拥挤度得分（0-25）+ 仓位集中度得分（0-25）+ 自我实现指数得分（0-25）+ 跨框架一致性得分（0-25），满分100分
- 压力等级：绿色（0-30）→ 黄色（30-50）→ 橙色（50-70）→ 红色（70-100）
- 对Layer 5执行的影响：绿色正常执行，黄色Alpha偏离降至75%，橙色降至50%，红色降至25%并强制触发人类裁决接口

**范式稳定性监测（4.5.3）**：
- 三大范式稳定性代理指标：跨资产相关性结构断裂、政策框架措辞质变、全球资本流动结构突变
- 范式稳定性综合判定：范式稳定（0个代理指标触发）→ 范式动摇（1个代理指标触发）→ 范式转移（≥2个代理指标触发）
- 对框架执行的影响：稳定正常执行，动摇全部信号强度×0.7，转移停止输出新的Alpha偏离信号，仅维持Beta基准

**人类最终裁决接口（4.5.4）**：
- 触发条件（满足任一即触发）：反身性压力综合得分>70（红色）、范式稳定性判定为"范式转移"、框架信号与Layer 4.5压力计方向持续冲突（连续3周信号方向正确但压力计维持橙色以上）
- 触发后流程：自动暂停Alpha偏离执行力度降至25%、状态呈现、定性类比、人类裁决、记录归档

**信号修正合成规则（4.5.5）**：
- 最终信号强度 = Layer 4 初始强度 × 逻辑生命周期修正系数 × 反身性压力修正系数 × 范式稳定性修正系数
- 逻辑生命周期修正系数：萌芽期1.0 / 扩散期0.9 / 共识期0.6 / 透支期0.3 / 崩塌期0
- 反身性压力修正系数：绿色1.0 / 黄色0.75 / 橙色0.50 / 红色0.25
- 范式稳定性修正系数：稳定1.0 / 动摇0.7 / 转移0（仅维持Beta）

**输出**：反身性压力综合得分（0-100分 + 四项子指标得分）、范式稳定性判定（稳定/动摇/转移）、修正后信号强度

### 3.8 Layer 5：资产配置与权重输出
**目标**：将信号转化为可执行的投资组合权重。

**步骤**：
1. 计算中国版All Weather Beta基准权重（A股15% + 中国国债35% + 南华工业品15% + 黄金15% + 美元资产20%）
2. 根据风险预算约束校验Beta层权重（组合年化波动率≤8%）
3. 根据衰减调整后强度≥50分位的信号，计算Alpha偏离幅度
4. 执行风险预算约束（单信号最大偏离±30%、单资产最大权重偏离±50%、组合年化波动率上限12%、最大回撤硬性上限15%、组合杠杆上限1.5x）
5. 计算行业配置得分（周期匹配度×3 + 景气趋势 + 政策催化）

**输出**：Beta基准权重、Alpha偏离权重、最终资产配置权重、行业配置建议、风险预算校验结果

## 4. 判断规则

### 4.1 方向（direction）判断规则

| 信号类型 | bullish（看多） | bearish（看空） | neutral（中性） |
|---|---|---|---|
| 增长预期差 | 实际增长强于市场定价（CAI - 利率隐含增长 > 0.5σ） | 实际增长弱于市场定价（CAI - 利率隐含增长 < -0.5σ） | |z-score| ≤ 0.5σ，或数据不足 |
| 通胀预期差 | 实际通胀强于定价（通胀得分 - 隐含通胀 > 0.5σ） | 实际通胀弱于定价（通胀得分 - 隐含通胀 < -0.5σ） | |z-score| ≤ 0.5σ |
| 政策预期差 | 政策超预期宽松（实际政策力度 - IRS隐含政策 > 0.5σ） | 政策超预期收紧（实际政策力度 - IRS隐含政策 < -0.5σ） | |z-score| ≤ 0.5σ |
| 外部预期差 | 压力小于担忧（实际外部压力 - 市场定价压力 < -0.5σ） | 压力大于担忧（实际外部压力 - 市场定价压力 > 0.5σ） | |z-score| ≤ 0.5σ |
| 周期定位 | 复苏/过热象限 | 滞胀/衰退象限 | 象限边界模糊（CAI在0附近波动） |
| 汇率方向 | USD/CNH方向得分 < -1.0σ（升值趋势确认） | USD/CNH方向得分 > +1.0σ（贬值趋势确认） | 方向得分在±0.5σ之间（震荡/无方向） |

> **注意**：多个信号冲突时，执行"周期一致性校验"规则（全球周期优先、恐慌状态直接下调、中美分化仅输出结构性信号）。

### 4.2 置信度（confidence）计算规则

**计算公式**：
```
confidence = 基础置信度 × 数据完整性系数 × 信号一致性系数
```

其中：
- **基础置信度**：根据信号强度评分确定
  - 强度评分 ≥ 80%分位：基础置信度 = 0.9
  - 强度评分 60%-80%分位：基础置信度 = 0.7
  - 强度评分 50%-60%分位：基础置信度 = 0.5
  - 强度评分 < 50%分位：基础置信度 = 0.3（通常输出neutral）
- **数据完整性系数**：1.0（无缺失）~ 0.8（部分缺失）
- **信号一致性系数**：1.0（所有信号同向）~ 0.6（信号冲突严重）

**置信度区间含义**：
- 0.8-1.0：证据充分且方向一致，可执行方向性判断
- 0.6-0.8：证据较充分但有少量不确定，可执行的但控制仓位
- 0.4-0.6：证据有限或互相矛盾，仅做结构性配置
- < 0.4：通常使用 `neutral` 并人工复核

### 4.3 风险等级（risk_level）判断规则

| risk_level | 判定条件 | 对信号的影响 |
|---|---|---|
| **low** | 无异常事件，所有指标正常发布，信号方向一致 | 暂不改变主要判断 |
| **medium** | 轻度冲击（VIX单日攀升<10点，或沪深300单日跌幅<3%，或北向资金单日净流出<100亿） | 维持常规分析流程，仅调整信号强度（下调1-2分） |
| **high** | 重度冲击（VIX单日攀升≥10点或VIX>30，或沪深300单日跌幅≥5%，或北向资金单日净流出≥200亿） | 暂停常规周期判断，优先跟踪全球流动性、VIX恐慌指数；仅输出观望/风险预警信号 |

**特殊高风险情形**（无论其他条件如何，直接设为 `high`）：
- Sahm Rule触发（≥0.5）
- 流动性危机（银行间市场资金面异常紧张、美元流动性枯竭）
- 政策发生重大变化（如央行更换行长、美联储紧急降息）
- 海外市场出现黑天鹅事件

### 4.4 人工复核触发规则

以下情形必须将 `needs_human_review` 设为 `true`：

1. **高风险事件**：risk_level = "high"
2. **数据缺失**：必填输入缺失超过3个指标
3. **信号冲突**：多个信号方向矛盾，且无法用"周期一致性校验"规则解决
4. **低频数据插值**：使用了插值数据（如季度GDP插值到月度），需在 `uncertainties` 中标注
5. **框架未覆盖情形**：遇到框架手册未明确的情形（如新问题1/2/5/7/15）

### 7.6 Layer 4：预期差信号引擎

#### 核心思想：预期差驱动
Alpha的真正来源是**预期差**——实际发生与市场定价之间的偏差。

#### 三类预期差信号

**信号类型A：高频意外指数**
- 跟踪每次数据发布的实际值与市场共识的差，捕捉短期方向冲击
- 中国CESI（China Economic Surprise Index）：正值=中国数据持续超预期
- 美国Citi Economic Surprise Index：正值=美国数据持续超预期

**信号类型B：基本面vs市场定价（核心Alpha来源）**

| 预期差类型 | 计算公式 | 正信号含义 | 对应交易 |
|---|---|---|---|
| 增长预期差 | 中国CAI − 利率隐含的增长预期（10Y-2Y利差z-score） | 实际增长强于市场定价→市场低估增长 | 做多增长资产（A股、商品） |
| 通胀预期差 | 中国通胀得分 − 隐含通胀预期（通胀挂钩债/Breakeven z-score） | 实际通胀强于定价→市场低估通胀 | 做多商品/做空债券 |
| 政策预期差 | 实际政策力度（降息/降准/净投放综合z-score）− IRS隐含的政策预期 | 政策超预期宽松→市场未充分定价 | 做多A股/信用债 |
| 外部预期差 | 实际外部压力（FCI美国分项+USD/CNH方向得分）− 市场定价的压力（CNH远期点+信用利差z-score） | 压力小于担忧→市场过度恐慌 | 做多港股/外资重仓股 |

**信号类型C：跨国预期差**

| 跨国预期差信号 | 构建逻辑 | 对应押注 |
|---|---|---|
| 中美增长预期差 | (中国CAI − 中国市场定价增长) − (美国CAI − 美国市场定价增长) | A股vs美股（相对价值）、中债vs美债 |
| 中美通胀预期差 | (中国通胀得分 − 中国隐含通胀) − (美国通胀得分 − 美国隐含通胀) | USD/CNH方向、相对债券收益率利差交易 |
| 中美政策预期差 | (中国政策力度 − 中国隐含政策) − (美国政策力度 − 美国隐含政策) | A股vs美股、CNH vs USD |
| 中美流动性差 | 中国FCI − 美国FCI（标准化后） | 全球资本流向、新兴市场偏好 |
| 中美信用周期差 | 中国信用脉冲 − 美国信用脉冲 | 相对周期股表现、铜价方向 |
| 美元-大宗联动信号 | Layer 2.5全球宏观三角的状态切换 | 整体风险资产偏好的整体偏置 |

#### 信号强度与有效性评级

| 评级 | 信号强度 | 历史分位 | 执行指引 |
|---|---|---|---|
| 8-10分 | 强信号 | ≥80%分位 | 可执行方向性判断，允许超越基准偏移 |
| 5-7分 | 有效信号 | 50%-80%分位 | 可执行但控制仓位，偏移不超过基准的±15% |
| 0-4分 | 弱信号 | <50%分位 | 仅做结构性配置，不做方向性押注 |

#### 信号衰减与复核规则

**信号有效期衰减规则**：

| 信号类型 | 初始有效期 | 衰减方式 | 说明 |
|---|---|---|---|
| 类型A：高频意外 | 2-4周 | 每周衰减25% | 数据发布冲击衰减最快 |
| 类型B：基本面vs定价 | 1-3个月 | 每月衰减20% | 基本面定价修正具有持续性 |
| 类型C：跨国预期差 | 1-3个月 | 每月衰减15% | 跨国周期错位持续性最长 |
| 前瞻信号 | 最长1个月 | 硬性到期 | 前瞻信号有效期不超过1个月，到期强制降级 |

> **待确认（问题7）**：信号衰减的具体形式（指数衰减 or 线性衰减）未明确声明，需专家4组确认。框架示例中暗示是指数衰减（乘以剩余百分比），但需明确公式。

**信号发布后跟踪规则**：
- 连续2周反向：触发复核——检查 Layer 1实际状态是否已变化、Layer 3市场定价是否已修正
- 连续3周反向：强制降级——信号强度评分降至 ≤50分位，转为弱信号

**信号强制清零规则**（不再参与 Layer 5权重计算）：
1. 信号有效期到期且未续期
2. 连续3周反向且复核确认基本面已反转
3. 预期差从显著水平回落至 |z-score| < 0.5σ

#### Layer 4 输出形态

| 输出项 | 形态 | 频率 |
|---|---|---|
| 3类预期差信号方向与强度 | 方向（正/负）+ 强度（百分位）+ 边际变化（↗/↘/---） | 周度 |
| 信号相关性矩阵 | 相关系数热力图 | 季度 |
| 信号复核记录 | 触发原因 + 复核结论 + 后续行动 | 周度（触发时） |
| 不相关信号源筛选结果 | 10-15个低相关信号列表 + 组合夏普估计 | 季度 |

### 7.6 Layer 4.5：反身性与元认知层（V3.1新增）

#### 设计依据

- 基于索罗斯的反身性理论：市场参与者的行为本身也改变市场
- 当越来越多资金采用相似的宏观分析框架和信号时，这些信号本身可能改变市场的运行方式
- 本层为框架装上一层"元认知"，回答"当前框架的信号是否仍然可靠？"

#### 三层元认知功能

| 功能 | 解决的问题 | 对应模块 |
|---|---|---|
| **反身性压力计** | 信号本身是否因共识拥挤而失效？ | 7.6.2 |
| **范式稳定性监测** | 当前环境是否仍在框架公理的适用范围内？ | 7.6.3 |
| **逻辑生命周期监控** | 当前主导逻辑处于生命周期的哪个阶段？ | 7.6.4 |

#### 7.6.2 反身性压力计

监测"共识的拥挤程度"，在大众同向共振时自动降低信号的执行力度。

**四大压力指标**：

| 指标 | 计算方式 | 预警阈值 | 极端阈值 | 信号含义 |
|---|---|---|---|---|
| **信号拥挤度** | 采用与框架同方向的策略资金规模 / 市场总容量（以CFTC非商业持仓 + ETF资金流 + 北向资金综合估算） | > 50% | > 70% | 共识交易占比过高，Alpha消失 |
| **仓位集中度** | CFTC非商业净头寸z-score（5年滚动） | \|z\| > 1.5 | \|z\| > 2.5 | 极端持仓，反身性风险高 |
| **信号自我实现指数** | 信号发布后价格变动中"预期驱动"占比 = （价格变动幅度 − 基本面变化可解释幅度）/ 价格变动幅度 | > 60% | > 80% | 价格变动主要由预期推动而非实际变化 |
| **跨框架一致性** | 主流卖方/买方策略报告与框架信号的吻合度（抽样统计前10大卖方 + 前5大买方） | > 70% | > 90% | 太多人看到同样的信号 |

**压力计合成**：

反身性压力综合得分 = 信号拥挤度得分（0-25）+ 仓位集中度得分（0-25）+ 自我实现指数得分（0-25）+ 跨框架一致性得分（0-25），满分100分。

| 综合得分区间 | 压力等级 | 对Layer 5执行的影响 |
|---|---|---|
| 0-30 | 绿色（低压力） | 正常执行，信号强度不修正 |
| 30-50 | 黄色（中压力） | Alpha偏离执行力度降至 **75%** |
| 50-70 | 橙色（高压力） | Alpha偏离执行力度降至 **50%**，触发人工复核 |
| 70-100 | 红色（极高压） | Alpha偏离执行力度降至 **25%**，强制触发人类裁决接口（7.6.4） |

**更新频率**：周度更新。红色状态时转为日度更新。

#### 7.6.3 范式稳定性监测

反身性压力计处理的是"常规范式内的共识拥挤"。范式稳定性监测处理的是更深层的问题：**当前环境是否仍在框架公理的适用范围内？**

**三大范式稳定性代理指标**：

| 代理指标 | 监测方法 | 预警条件 | 信号含义 |
|---|---|---|---|
| **跨资产相关性结构断裂** | 计算铜金比与PMI、利差与增长、ERP与增长等6对核心相关性的5年滚动相关系数；当任一相关性在6个月内从 \|ρ\| > 0.6 降至 \|ρ\| < 0.2 | 对相关性同时断裂 ≥ 2 | 历史定价关系失效，框架的统计基础动摇 |
| **政策框架措辞质变** | 追踪美联储/央行政策框架的措辞变化，区分"边际调整"（已有4.4.7覆盖）与"根本性重新定义"（如通胀目标制→平均通胀目标制、央行直接购买股票/ETF、收益率曲线控制） | 4.4.7 | 政策框架出现质变 |
| **全球资本流动结构突变** | 追踪跨境资本流向与5年历史模式（BIS/TIC数据）的背离程度，做5年滚动z-score | 背离持续3个月以上且 \|z\| > 2.0 | 全球资本配置逻辑正在发生结构性变化 |

**范式稳定性综合判定**：

| 状态 | 触发条件 | 对框架执行的影响 | 修正系数 |
|---|---|---|---|
| **范式稳定** | 0个代理指标触发 | 正常执行 | × 1.0 |
| **范式动摇** | 1个代理指标触发 | 全部信号强度 × 0.7，新增投资叙事标记"范式不确定"，Alpha偏离幅度减半 | × 0.7 |
| **范式转移** | ≥ 2个代理指标触发 | **框架进入"观察模式"**：停止输出新的Alpha偏离信号，仅维持Beta基准；强制触发人类裁决接口（7.6.4）；报告开头标注"范式转移预警——框架信号可靠性下降" | × 0（仅Beta） |

**与4.4异常场景的区别**：4.4处理的是"已知类型的极端事件"（VIX飙升、流动性危机），本模块处理的是"框架适用性本身的质疑"——不是市场出了问题，而是**我们理解市场的方式可能出了问题**。

**更新频率**：月度更新，触发时即时评估。

#### 7.6.4 人类最终裁决接口

当反身性压力计或范式稳定性监测触发到严重程度时，不自动执行决策，而是向人类呈现观察结果，请求"最高智慧"的介入。

**触发条件（满足任一即触发）**：

1. 反身性压力综合得分 > 70（红色）
2. 范式稳定性判定为"范式转移"
3. 框架信号与Layer 4.5压力计方向持续冲突（连续3周信号方向正确但压力计维持橙色以上）

**触发后流程**：

| 步骤 | 动作 |
|---|---|
| 1. 自动暂停 | Alpha偏离执行力度降至25%，新信号暂停发布 |
| 2. 状态呈现 | 向框架负责人呈现：当前信号、压力计读数、范式稳定性状态、历史类比（如有） |
| 3. 定性类比 | 面对从未见过的冲击时，不强行量化，通过历史定性类比判断风险性质（参考案例库中的模式匹配） |
| 4. 人类裁决 | 框架负责人做出决策：①恢复执行 ②维持观察模式 ③手动调整信号方向 |
| 5. 记录归档 | 裁决过程与结论完整记录，纳入框架迭代素材 |

**关键原则**：

人类裁决接口不是否定框架，而是在框架信号可靠性下降时的安全网。它确保"最高智慧"在关键时刻介入，而非让系统在不确定环境中自行决策。

#### 7.6.5 信号修正合成规则

Layer 4输出的初始信号强度，需经Layer 4.5三重修正后，方可传入Layer 5：

**最终信号强度 = Layer 4 初始强度 × 逻辑生命周期修正系数（4.6.3）× 反身性压力修正系数（7.6.2）× 范式稳定性修正系数（7.6.3）**

| 修正来源 | 修正系数取值 |
|---|---|
| 逻辑生命周期（4.6.3） | 萌芽期 1.0 / 扩散期 0.9 / 共识期 0.6 / 透支期 0.3 / 崩塌期 0 |
| 反身性压力（7.6.2） | 绿色 1.0 / 黄色 0.75 / 橙色 0.50 / 红色 0.25 |
| 范式稳定性（7.6.3） | 稳定 1.0 / 动摇 0.7 / 转移 0（仅维持Beta） |

**示例**：某信号初始强度80%（强信号），逻辑生命周期处于"共识期"（×0.6），反身性压力"黄色"（×0.75），范式"稳定"（×1.0），则修正后强度 = 80% × 0.6 × 0.75 × 1.0 = **36%**（弱信号，不产生Alpha偏离）。

**修正后信号强度与Alpha偏离的对应关系**：

| 修正后强度 | 信号等级 | Alpha偏离 |
|---|---|---|
| ≥ 80% | 强信号 | 偏离系数 0.3 |
| 60%-80% | 中强信号 | 偏离系数 0.2 |
| 50%-60% | 弱信号 | 偏离系数 0.1 |
| < 50% | 无效信号 | 不产生Alpha偏离 |

> **关键规则**：当框架的预期差信号方向与市场主流共识方向一致时（4.6.2第四行），无论绝对强度如何，信号强度至少降至**60%**——这是反身性最直接的体现：**如果所有人都在交易同一个预期差，它就不再是预期差了。**

#### 7.6.6 Layer 4.5 输出形态

| 输出项 | 形态 | 频率 |
|---|---|---|
| 反身性压力综合得分 | 0-100分 + 四项子指标得分 | 周度（红色时日度） |
| 范式稳定性判定 | 稳定/动摇/转移 | 月度（触发时即时） |
| 逻辑生命周期判定 | 萌芽期/扩散期/共识期/透支期/崩塌期 | 月度 |
| 修正后信号强度 | Layer 4初始强度 × 三重修正系数 | 周度 |

**数据流向**：Layer 4（预期差信号方向 + 初始强度）→ **Layer 4.5（反身性压力修正 + 范式稳定性修正 + 逻辑生命周期修正 → 修正后信号强度）** → Layer 5（修正后信号驱动的资产配置权重）

### 7.7 Layer 5：资产配置与权重输出

#### 核心架构：Beta + Alpha 双层
- **Beta层（All Weather风险平价）**：保证信号失效时仍能存活
- **Alpha层（预期差信号驱动）**：让信号有效时跑出超额

#### 中国版All Weather Beta基准

| 资产类别 | 对应象限 | 工具 | 基础权重 |
|---|---|---|---|
| A股（沪深300+中证500） | 复苏/过热 | 股指期货（IF/IC）、宽基ETF | 15% |
| 中国国债（10年期为主） | 衰退/流动性陷阱 | 国债期货（T）、利率债ETF | 35% |
| 南华工业品 | 过热/通胀环境 | 商品期货组合（铜、原油、铁矿石、螺纹钢） | 15% |
| 黄金 | 滞胀/危机 | 黄金期货（AU）、黄金ETF（518880） | 15% |
| 美元资产（美债+QDII） | 外部对冲 | QDII基金、港美股ETF（513500/513100） | 20% |

**权重确定**：
1. 计算各资产3年滚动年化波动率σᵢ
2. 基础权重wᵢ = (1/σᵢ) / Σ(1/σⱼ)
3. 校验组合年化波动率≤8%（若超标，等比例缩降权重）

#### Alpha偏离的生成

| 信号 | 方向 | Alpha偏离 |
|---|---|---|
| 中国增长预期差+ | 实际增长强于定价 | A股↑、商品↑、利率债↓ |
| 中国通胀预期差+ | 实际通胀强于定价 | 商品↑、上游股↑、利率债↓ |
| 中国政策预期差+ | 政策超预期宽松 | A股↑、信用债↑、地产链↑ |
| 外部压力预期差− | 压力小于担忧 | 港股↑、外资重仓股↑、黄金↓ |
| 中美增长预期差+ | 中国相对增长更被低估 | A股vs美股（超配A股）、中债vs美债（超配中债） |
| 全球紧缩三角 | Layer 2.5三角定位为全球紧缩 | 整体降低风险资产、增加黄金 |
| 铜金比极端低位+增长信号好 | 铜被过度低估+增长确认 | 周期股vs防御股（超配周期股） |
| USD/CNH贬值压力↑ | 人民币贬值预期增强 | 港股↓、外资重仓股↓、出口股↑、黄金↑ |

**Alpha偏离计算规则**：

1. 每个有效信号产生一个偏离方向，**信号强度取 Layer 4.5 修正后的最终强度**（而非 Layer 4 初始强度）

2. **最终信号强度 = Layer 4 初始强度 × (1 − 累计衰减率) × 逻辑生命周期修正 × 反身性压力修正 × 范式稳定性修正**

   - 示例：初始强度80%，B类信号已发布2个月（累计衰减40%），逻辑"扩散期"（×0.9），反身性"绿色"（×1.0），范式"稳定"（×1.0），最终强度 = 80% × 0.6 × 0.9 × 1.0 × 1.0 = 43.2%，低于50%阈值，不产生Alpha偏离

3. 偏离幅度 = 基准权重 × 信号强度系数（修正后强度≥80%→系数0.3；60%-80%→0.2；50%-60%→0.1）

4. 多个信号对同一资产产生同向偏离时，偏离幅度相加，但受风险预算约束上限限制

5. 多个信号对同一资产产生反向偏离时，取净值

> **关键规则**：当框架的预期差信号方向与市场主流共识方向一致时（4.6.2第四行），无论绝对强度如何，信号强度至少降至**60%**——这是反身性最直接的体现：**如果所有人都在交易同一个预期差，它就不再是预期差了。**

#### 行业景气映射

**A股行业轮动映射表**：

| 周期阶段 | 领先受益行业 | 防御性行业 | 规避行业 |
|---|---|---|---|
| 复苏周期 | 顺周期（钢铁/有色/建材）、银行、非银金融、可选消费（汽车/家电） | 消费、医药 | 高估值成长、纯主题板块 |
| 扩张周期 | 新能源（储能/光伏）、工业机械、半导体设备、出口链 | 银行（估值回归稳定） | 强周期资源类（高位兑现） |
| 滞胀周期 | 上游资源（煤炭/石油/铜）、农业、黄金 | 公用事业、消费必选 | 高杠杆成长、地产链 |
| 衰退周期 | 公用事业（电力/水务）、必选消费（食品饮料）、医疗服务 | 高分红红利资产 | 强周期/可选消费/科技成长 |

**行业配置量化评分公式**：
- 行业配置得分 = 周期匹配度（0或1）× 3 + 景气趋势（上行+2/持平+1/下行+0）+ 政策催化（有+2/无+0）
- ≥5分：一级配置（超配）
- 3-4分：二级配置（标配）
- ≤2分：三级配置（低配/规避）

#### 风险预算约束

| 约束项 | 阈值 | 校验频率 | 说明 |
|---|---|---|---|
| 单信号最大偏离 | 基准的 ±30% | 每周 | 任一信号对单一资产的最大偏离不超过基准权重的30% |
| 单资产最大权重偏离 | 基准的 ±50% | 每周 | Alpha偏离后，单资产最终权重不超过基准的150%、不低于50% |
| 组合年化波动率上限 | 12% | 每月 | 基于过去3年协方差矩阵估算，超标时等比例缩降Alpha偏离 |
| 最大回撤硬性上限 | 15% | 实时 | 触及15%回撤时，Alpha偏离全部清零，回归纯Beta基准 |
| 组合杠杆上限 | 1.5x | 每周 | 所有资产权重之和不超过150%，防止过度集中 |

> **说明**：Beta层权重确定时已校验组合年化波动率≤8%，此处12%是Alpha偏离后的上限。

## 8. 核心指标体系

### 8.1 指标选取核心原则
1. **领先性优先原则**：优先选取领先于经济周期、领先于资产价格的指标
2. **可量化原则**：指标需有明确的数值、可标准化统计
3. **口径统一原则**：所有指标均固定统计口径、频率、来源
4. **强相关性原则**：指标需与宏观周期、A股大类资产价格高度相关
5. **可获取原则**：指标数据需公开可获取、及时更新
6. **频率分层原则**：按日/周/月/季频分层设置指标
7. **AI可执行原则**：所有指标规则可代码化、无模糊空间
8. **合规性原则**：所有数据来源均为官方合规渠道

### 8.2 标准化指标清单（详见框架手册3.2节）
- **3.2.0**：另类资产核心指标（CRB指数、WTI原油、COMEX黄金、南华商品指数）

### 8.3 指标口径强制规范
- **PMI类指标**：全球/国内统一采用官方季调后口径
- **同比类指标**：除特殊说明外，均采用"季调后3个月移动平均"口径
- **利率/汇率/波动率类指标**：均采用"日度均值→月度滚动平均"口径
- **剪刀差/占比类指标**：均采用"当月原始数据计算"，不做移动平均

### 8.5 指标有效性回测与迭代规则
1. **半年度全量回测：** Skill输出小组每半年对所有核心指标的领先性、相关性、胜率进行全历史回测，剔除胜率低于60%的弱有效指标，补充高胜率有效指标；
2. **季度优化校准：** 大周期融合小组每季度根据复盘结果，对指标阈值、口径进行校准，校准方案需经过回测验证、全组评审后方可更新；

## 9. 异常场景处理规则

### 9.1 黑天鹅事件分级处理

| 分级 | 量化判定标准（满足任意2条即认定） | 处理规则 |
|---|---|---|
| 轻度冲击 | 1. VIX单日攀升<10点；2. 沪深300单日跌幅<3%；3. 北向资金单日净流出<100亿 | 维持常规分析流程，仅调整信号强度（下调1-2分）；不否定已有周期定位 |
| 重度冲击 | 1. VIX单日攀升≥10点或VIX>30；2. 沪深300单日跌幅≥5%；3. 北向资金单日净流出≥200亿或连续5日净流出；4. 美元指数单日波动>1.5% | 暂停常规周期判断，优先跟踪全球流动性、VIX恐慌指数；仅输出观望/风险预警信号 |

### 9.2 政策超预期突变
- **常规政策调整**：按原规则等待1-2个月核心指标验证，再按4.2拐点标准判定周期切换
- **历史级强政策对冲**（量化标准见3.3第10条）：立即启动高频指标跟踪，可临时调整信号方向，同步等待月度数据验证

### 9.3 数据大幅修正
按修正后的数据重新执行4.2拐点判定，仅调整未来信号，不追溯过往结论。

### 9.4 中美周期严重分化
美联储大幅加息，国内大幅宽松（或反之），信号有效性直接下调至**5分及以下**，仅可输出结构性配置信号或观望信号，等待周期收敛。

### 9.5 流动性危机
银行间市场资金面异常紧张、美元流动性枯竭，无论周期定位如何，直接输出看空/观望信号，优先提示风险。

### 9.6 Sahm Rule衰退预警
- **触发条件**：Sahm Rule读数≥0.5
- **处理规则**：
  - 无论其他指标如何，强制输出"美国衰退预警"标志
  - 美国基本面模块标签强制判为"下行"
  - 美股信号下调1档（看涨→观望，观望→看空）

### 9.7 政策预期打分卡
**政策预期打分卡（0-10分）**：

| 打分项 | 评分标准（每项0-2分） | 数据来源 |
|---|---|---|
| **官方媒体措辞变化** | 0分：措辞不变；1分：出现"积极""有力""加大"等边际强化措辞；2分：出现"超常规""历史性""全面"等强转向措辞 | 新华社、人民日报头版/社论，AI NLP关键词抓取 |
| **国常会频次与议题** | 0分：常规频次（每周1次），无经济议题；1分：频次增加至每周2次以上，或经济议题占比>50%；2分：连续2次以上聚焦经济，且措辞边际强化 | 中国政府网、国常会公告 |
| **央行货政报告措辞** | 0分：维持"稳健""灵活适度"等中性措辞；1分：出现"加大力度""更加有力"等边际宽松措辞；2分：出现"充分发力""全面支持"等强宽松措辞 | 央行季度货币政策执行报告 |
| **高层会议信号** | 0分：无相关信号；1分：政治局会议提及"加大宏观调控力度"；2分：政治局会议或中央经济工作会议明确提出"超常规"政策方向 | 新华社通稿 |
| **政策落地节奏** | 0分：无新政策落地；1分：单次政策落地（如降息10BP、降准25BP）；2分：连续2次以上政策落地，或单次力度超预期 | 央行、财政部公告 |

**打分卡触发规则**：
1. **总分 ≥ 7分**：判定为"**政策强转向预期**"，允许在4.2拐点确认标准尚未满足的情况下，提前1个月输出前瞻信号，信号方向与政策转向方向一致；
2. **总分 5-6分**：判定为"**政策边际宽松预期**"，信号有效性+1分，但不触发前瞻信号；
3. **总分 ≤ 4分**：无政策转向预期，不调整信号。

> **待确认**：政策预期打分卡的具体执行频率（每月？每季？）未明确，需专家4组确认。
