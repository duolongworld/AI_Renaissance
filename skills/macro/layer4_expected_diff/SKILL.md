# Layer 4: 预期差信号引擎

## 执行方式

- [ ] 数值计算
- [x] 智能分析（需要LLM解读预期差含义、交易含义）
- [ ] 混合模式

## 适用范围

计算"实际状态vs市场定价"的偏差，为 Layer 5 资产配置提供信号输入。

## 信号类型

### 类型A：高频意外指数

- 中国CESI：正值=中国数据持续超预期
- 美国CESI：正值=美国数据持续超预期

### 类型B：基本面vs市场定价

| 信号 | 计算 | 正信号含义 |
|------|------|-----------|
| 增长预期差 | CAI - 利率隐含增长 | 实际增长强于定价 |
| 通胀预期差 | 通胀得分 - 隐含通胀 | 实际通胀强于定价 |

### 类型C：跨国预期差

| 信号 | 含义 |
|------|------|
| 中美增长预期差 | A股vs美股相对价值 |
| 中美流动性差 | 全球资本流向 |

## 信号衰减规则

| 类型 | 初始有效期 | 衰减方式 |
|------|-----------|----------|
| A类 | 2-4周 | 每周25% |
| B类 | 1-3个月 | 每月20% |
| C类 | 1-3个月 | 每月15% |

## 复核规则

- 连续2周反向 → 触发复核
- 连续3周反向 → 强制降级至弱信号
