# 七步链框架

## Step0 红色预警扫描

| 类别 | 红色预警 | 处理 |
|---|---|---|
| 收入端 | 收入确认政策变更、销售收现/营收长期低于 0.8、应收增速持续高于营收增速、关联方销售异常 | 至少降一档 direction，核心预警触发人工复核 |
| 费用端 | 费用资本化率异常、折旧摊销年限长于同行、财务费用非现金项目占比高、大额费用跨期 | 要求披露调整前后口径 |
| 资产负债端 | 其他应收暴增、预付款项长期挂账、存货大增但跌价不足、商誉高且未减值 | 高风险项进入 `meta.red_flags` |
| 治理端 | 财务重述、大股东高比例质押、资金占用、少数股东损益异常 | 核心治理预警触发 `needs_human_review` |

## Step1 利润质量

| 指标 | 公式 | 判断 |
|---|---|---|
| 现金利润比 | 经营活动现金流净额 / 归母净利润 | >1.2 优秀；1.0-1.2 良好；0.8-1.0 一般；0.5-0.8 警惕；<0.5 危险 |
| 销售收现/营收 | 销售商品、提供劳务收到的现金 / 营业收入 | 1.0 附近正常；<0.8 危险；>1 需区分增值税和前期回款 |
| 应收增速 vs 营收增速 | 应收变化率与营收变化率对比 | 应收慢于营收为积极；应收快于营收为警惕 |

## Step2 现金流匹配

| 指标 | 公式 | 判断 |
|---|---|---|
| 采购付现/营业成本 | 购买商品、接受劳务支付的现金 / 营业成本 | 长期严重偏离需解释供应链付款节奏 |
| 经营现金流口径校正 | 经营现金流 +/- 租赁准则和非经常项目影响 | 新租赁准则下需还原可比口径 |
| 收现与付现配比 | 销售收现趋势与采购付现趋势 | 同向更可信，背离需解释 |

## Step3 需求真实性

| 信号 | 积极解释 | 风险解释 |
|---|---|---|
| 应收下降 + 收现增强 | 客户付款意愿强，需求真实 | 需确认是否保理或一次性回款 |
| 合同负债增速 > 营收增速 | 新订单领先交付 | 需确认合同负债性质 |
| 存货上升 + 在建上升 | 为未来交付备产 | 若合同负债下降，可能滞销 |
| 其他应收稳定或下降 | 资金占用压力低 | 暴增需排查关联方和大股东 |

## Step4 资本开支

| 指标 | 公式 | 判断 |
|---|---|---|
| 购建现金流 | 购建固定资产、无形资产和其他长期资产支付的现金 | 直接反映当期投入力度 |
| 在建/固定/购建三联动 | 三项同时增长 | 真扩产信号更强 |
| capex/折旧覆盖比 | 购建现金流 / 折旧摊销 | >1 覆盖老化；制造业建议 >1.5；<1 长期收缩 |
| 固定资产周转率 | 营业收入 / 平均固定资产净额 | 走高代表产能趋紧，走低代表闲置风险 |

## Step5 债务与利率敏感性

| 指标 | 公式 | 判断 |
|---|---|---|
| 净债务率 | (有息负债 - 货币资金) / 股东权益 | 趋势上升需与 capex 和订单匹配 |
| 短债占比 | 短期有息负债 / 有息负债 | 短债过高意味着再融资风险 |
| 财务费用/营业利润 | 财务费用 / 营业利润 | >20% 高风险；10%-20% 需监控；<10% 相对安全 |
| 调整后财务费用率 | 剔除租赁负债利息、未确认融资费用摊销、一次性汇兑影响后 / 营收 | 用于识别真实融资负担 |

## Step6 扩张质量

| 配对 | 良性信号 | 风险信号 |
|---|---|---|
| 资产增速 vs 收入增速 | 资产合理领先收入 | 资产远快于收入，可能过度投资 |
| 资产增速 vs 合同负债增速 | 扩产有订单支撑 | 资产增长但合同负债下降 |
| 预付款项 vs 存货/在建 | 采购配合扩产 | 预付增长但资产不增，疑似资金占用 |
| 商誉/归母权益 | 低且稳定 | >30% 且标的业绩承压 |

## Step7 行业特殊口径

每次分析都必须显式执行 Step7。行业差异不应作为“免检理由”，只能用于调整阈值、口径和解释方式。详见 `industry_adaptations.md`。

## 附加工具

| 工具 | 用途 |
|---|---|
| 同行对标矩阵 | 毛利率、净利率、收现比、周转天数、资产负债率、研发费用率、capex/折旧 |
| 杜邦分析 | 把 ROE 拆成净利率、资产周转率、权益乘数 |
| 客户/供应链集中度 | 识别单一客户、单一供应商和回款依赖 |
| 前瞻指标 | Segment 结构、合同负债-营收剪刀差、人员变化、产能利用率替代估算 |
