# 已知陷阱

## 数字抽取陷阱

- PDF 表格列错位会把本期、上期、同比列混在一起。进入核心判断前必须回查原文。
- 单位常在“元/千元/万元/亿元”之间切换，统一单位后再计算。
- 合并报表和母公司报表不能混用。

## 科目混淆

- 预收款/合同负债是负债科目，代表客户先付款。
- 预付款项是资产科目，代表公司先付给供应商。
- 合同资产不是应收账款，需要结合收入确认和履约进度解释。

## 租赁准则

新租赁准则会把原经营租赁拆为折旧、利息和筹资现金流，可能导致经营现金流看起来更好。零售、连锁、航空等行业必须还原口径。

## 二手数据

研报和媒体可以提示方向，但不能替代公告原文。若只有二手数据，Signal 必须降置信并标记人工复核。

## 纸面利润

利润增长但现金利润比下降、应收增速高于营收、合同负债下降，通常意味着增长质量不足。不能只因净利润同比增长输出 bullish。

## 扩张误判

capex 放量不是天然好消息。只有当合同负债、订单、收现和产能利用率能支撑扩产时，才算良性扩张。
